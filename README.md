## 🌎 Choose Your Language / Escolha seu Idioma

🇧🇷 **Português (Brasil)**  
Bem-vindo ao repositório **NasmLinuxCodeLab**.

Este projeto contém a implementação prática dos exemplos e exercícios do livro **Assembly na Prática – Versão 1.0**, do professor Fernando Anselmo.

O objetivo é consolidar fundamentos de programação de baixo nível utilizando NASM em ambiente Linux, abordando conceitos como:

- Arquitetura x86 (32 bits)
- Manipulação de registradores
- Uso da pilha (stack)
- Chamadas de sistema (syscalls)
- Geração de executáveis ELF

Este repositório possui finalidade exclusivamente educacional e integra minha jornada de aprofundamento em linguagens de baixo nível.

[📖 Leia a documentação em Português](https://github.com/fmarqueseti/NasmLinuxCodeLab/blob/main/README_BR.md)

🇺🇸 **English (US)**  
Welcome to the **NasmLinuxCodeLab** repository.

This project contains practical implementations of the examples and exercises from the book *Assembly na Prática – Versão 1.0* by Fernando Anselmo.

Its purpose is to strengthen low-level programming fundamentals using NASM in a Linux environment, covering topics such as:

- x86 architecture (32-bit)
- Register manipulation
- Stack usage
- Linux system calls
- ELF executable generation

This repository is strictly educational and is part of my low-level systems programming study journey.

[📖 Read the English documentation](https://github.com/fmarqueseti/NasmLinuxCodeLab/blob/main/README_EN.md)
